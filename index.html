General

CSS flexbox is a concept that allows for ordering elements in rows or columns. It is great for repsonsive layouts!

Flex container

Adding display: flex to an element turns it into a flex container. This makes it possible to align any children of that element into rows or columns. flex-direction determines the direction of child elements:

  • row (default)
  • column
  • row-reverse
  • column-reverse

justify-content determines how child elements should be positioned along the main axis:

  • flex-start (default)
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

align-items determines how child elements should be positioned along the cross axis:

  • flex-start (default)
  • flex-end
  • center
  • stretch
  • baseline

By default, a flex container fits all child elements together. If the flex container is too small for the items inside, flex-wrap: wrap allows for wrapping the flex items.

Flex items

Flex items will shrink if the surrounding container is too small. By specifying a flex-shrink property for the items, it is possible to control how much the items will shrink in relation to each other.

Flex items will not automatically expand if the surrounding container is too big. But by specifying flex-grow it is possible to make that happen. Again, it is used to determine the behaviour of the flex items in relation to each other.

flex-basis sets the width of a flex item and is by default equal to the width of the element, but will overwrite width if set. Also, flex-basis is the width along the main axis, whereas width is the horizontal width.
